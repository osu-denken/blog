## CLI からの接続
Windows のターミナルから
```bash
ssh <ProxmoxのローカルIP>
```
を叩く

## Web からの接続
ブラウザにて`<ProxmoxのローカルIP:8006>` へアクセスしてユーザ名とパスワードを入力することで操作画面に入ることができる。

## ISO のアップロード
ブラウザから接続したProxmoxの操作画面にてアップロードしたいサーバのプルダウン∨を押してlocal を選択してISO イメージからアップロードできます。
